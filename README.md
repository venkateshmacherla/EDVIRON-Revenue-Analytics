# EDVIRON Revenue Analytics – Power BI Project

## Project Overview
This project contains an interactive Power BI dashboard built for revenue analytics and performance tracking.

## Files Included
- Power BI Report (.pbix)
- Excel Dataset (Source Data)
- PDF Report (Dashboard & Visualizations)
- PDF Page Explanation Note

## Tools Used
- Power BI
- Microsoft Excel
- DAX Measures
- Data Modeling
